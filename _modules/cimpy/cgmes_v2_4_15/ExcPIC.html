	:e1: Field voltage value 1 (E1).  Typical Value = 0. Default: 0.0

	:e2: Field voltage value 2 (E2).  Typical Value = 0. Default: 0.0

	:kp: Potential source gain (Kp).  Typical Value = 6.5. Default: 0.0
